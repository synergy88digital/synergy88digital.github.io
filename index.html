    Auth

    App authentication

    POST /baas/public/api/v1/apps/{app_id}/authenticate HTTP/1.1
Content-Type: application/json

{
    "body": {
        "email": "string",
        "password": "string"
    }
}

    HTTP/1.1 200 OK
Content-Type: application/json

{
    "token": "string",
    "token_issued_at": "string",
    "token_expired": "string"
}

    HTTP/1.1 401 Unauthorized

    HTTP/1.1 405 Method Not Allowed

    Parameters

    Name In Type Description
    body *  body AuthCredentials Authentication credentials
    api_key *  query string API Key
    api_secret *  query string API secret

    Responses

    Http code Type Description
    200 TokenData Return token
    401 no content Unauthorized
    405 no content Invalid input
